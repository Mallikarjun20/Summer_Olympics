# Summer_Olympics

Modules Used - pandas, numpy, collections, matplotlib

## Concepts Used Through the Project
* File Handling
* Importing Modules
* Basic Data Structures concepts like - DICTIONARY, LISTS etc
* Basic Data manipulation
